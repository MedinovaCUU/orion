6.4.5. Usuarios
En la Ilustración 6.7 se muestra la pantalla de creación y mantenimiento de los usuarios.
Pulse en el icono de nuevo para acceder a la creación de nuevos usuarios. Se activan los campos para introducir los 
datos del usuario.
• 
ID usuario: Introduzca un nombre para identificar al usuario en la aplicación.
• 
Nombre: Introduzca el nombre del usuario.
• 
Apellido: Introduzca el apellido del usuario.
• 
Contraseña: Introduzca una contraseña 
• 
Confirmación contraseña: Introduzca otra vez la misma contraseña para confirmar que se ha introducido 
correctamente.
Pulse grabar para guardar el usuario o deshacer para descartar el usuario o los cambios.
