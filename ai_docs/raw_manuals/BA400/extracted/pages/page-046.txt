4.5. Photometric control board- CIIM00050
Board located in the reading rotor. It contains a micro controller (U5) which controls the following elements:
• 
Reaction rotor cover detection (U7)
• 
Wash station motor (U14)
• 
Wash station start-up detection (U10)
• 
Wash station collision detection (U10)
• 
Photometric board connection
• 
Reaction rotor motor (U12)
• 
Reaction rotor start-up detection (U10)
• 
Reaction rotor encoder detection (13)
• 
Peltier control in thermostatting the reaction rotor (U18)
• 
Fan control for cooling the peltiers (U17). There are 3 fans with 3 wires.
• 
Reaction rotor thermostatting temperature sensor control (U15 and U16)
