4.2. Placa distribución - CIIM00047
La placa distribución es la encargada de realizar la conexión para cada una de las diferentes placas. Las conexiones 
se realizan por el bus CAN. El cable del bus distribuye la alimentación para cada placa y las señales CAN.
Las placa de distribución lleva serigrafiado en cada conector que subconjuntos se conectan.
