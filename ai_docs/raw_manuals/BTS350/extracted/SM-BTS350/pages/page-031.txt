4.11.4. - Precision
This allows successive readings to be made, changing the sample each time. If using the same fluid, it allows the precision 
of the appliance to be tested (replicability between identical samples).
When beginning the test, the instrument requests:
• 
Reading wavelenght.
• 
Stabilisation time.
• 
Sample volume.
• 
Number of intervals.
Press F1 to begin the test.
The appliance asks for the baseline to be entered and then as many samples as there are numbers of intervals programmed. 
The test statistical data are printed at the end: maximum absorbance, minimum absorbance, average and coefficiant of 
variation (CV).
4.11.5. - Trueness
This allows the accuracy of a sample with a known value to be checked. 
WARNING: To guarantee the test is reliable, the known value of the sample must have been determined with a very exact 
reference method. 
When beginning the test, the instrument requests:
• 
Reading wvaelenght.
• 
Stabilisation time.
• 
Sample volume.
