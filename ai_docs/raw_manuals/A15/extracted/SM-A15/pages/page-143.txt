7.2.6. Proceso de verificación del analizador
Este es un proceso estándar para verificar el funcionamiento correcto del instrumento después de realizar 
una intervención.
Este proceso se puede aplicar a las siguientes operaciones:
• 
Recepción de instrumentos (en la oficina, antes de la instalación)
• 
Mantenimiento
• 
Después de una reparación
El objetivo es estandarizar el proceso de verificación y los criterios de aceptación.
El tiempo de duración del proceso es: 1,5 horas.

Use reactivos y controles de Biosystems o códigos equivalentes, use siempre nuevos calibradores y con-
