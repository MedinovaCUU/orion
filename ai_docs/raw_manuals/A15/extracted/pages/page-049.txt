4.11. Información sobre el canal auxiliar
La parte trasera izquierda del analizador es donde se conectan los cables de comunicación con el orde-
nador. Hay dos posibles conexiones el COM1 y el COM2.
El COM1 es la conexión principal del analizador con el ordenador, para hacer funcionar el analizador esta 
conexión siempre tiene que estar presente. Tiene dos posibilidades de conexión:
 
A - Cable tipo USB
 
B - Cable tipo RS-232
Solamente se tiene que conectar un tipo de cable.
El conector marcado como COM2 es el conector auxiliar.
En esta posición sirve para conectar un cable a un segundo puerto de comunicaciones serie del ordenador. 
Dicho cable sirve para realizar una escucha de los diferentes estados internos del analizador. 
Para visualizar dicha información el usuario tiene que ejecutar una sesión del programa HiperTerminal de 
Windows y configurarlo con los siguientes parámetros:
Programa: Inicio\Todos los programas\accesorios\comunicaciones\hyperterminal
