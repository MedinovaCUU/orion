6.4.2. AC13371 - Sensor de la temperatura del rotor
Desmontaje:
1. Extraer la carcasa superior. Ver 6.1.5
2. Desconectar la manguera de la sonda de temperatura
3. Retirar el aislante térmico del sensor de temperatura
4. Extraer el sensor de temperatura (3)
Montaje:
1. Sustituya el sensor de temperatura
2. Retirar la silicona térmica del alojamiento y poner nueva silicona térmica en el extremo del nuevo sensor
3. Montar el nuevo sensor de temperatura
4. Realizar el ajuste de temperatura del rotor
